Project outline:
- Create a webpage with a 16x16 grid of square divs
- Divs must be created using JavaScript
- Use Flexbox to make the divs appear as a grid
- Set up a "hover" effect so that the divs change color when your mouse passes over them
- Add a button that when clicked you'll be prompted to enter a number between 0-100; the 16x16 grid will then disappear and be replaced with the new grid in the same exact space as before.
